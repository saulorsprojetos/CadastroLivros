Cadastro de Livros
------------------

Sistema de gerenciamento de livros desenvolvido em ASP.NET Core (.NET 8).

* Funcionalidades:

-> Cadastro de Livros;

-> Autores e Assuntos;

-> Formas de Compra com Preços;

-> Relatório em PDF de Livros por Autor;

-> Testes automatizados.

* Tecnologias
-> ASP.NET Core MVC;
  
-> Entity Framework Core;

-> SQL Server;

-> xUnit.

* Estrutura
-> CadastroLivros.Domain;
  
-> CadastroLivros.Infrastructure;

-> CadastroLivros.Web;

-> CadastroLivros.Tests.
